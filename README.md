## 公開方法
1. https://vercel.com にアクセスし、GitHub またはメールでログイン。
2. 「+ New Project」→「Deploy from your computer」をクリック。
3. このフォルダをドラッグ＆ドロップ。
4. 数十秒後に URL が発行されます（例：https://rapunzel.vercel.app）。
